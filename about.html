What's all this then?

Thank you for your question. The website itself, Kaleidoscopia, is where I plan to post a series of works-in-progress, rough drafts, scraps. Roman Candle, which is live now, is the first of these. While there is still plenty of polishing to be done, things to cut, sentences to make... better, I am really fond of this piece.

I built this site myself and am hosting the code on GitHub so feel free to check it out to see just how messy my coding can be. All the fun retro fonts, images, and css are fully the work of NES.css and I want to thank their team for making such a fun CSS framework easy to use!

If you have any questions, edit suggestions, thoughts on the piece, comments about the site, or just generally want to say hi, feel free to email me at andrew.beasley.kc@gmail.com. Thanks so much for taking the time to read my work!
